# InnovateIT School — Frontend

Static HTML/CSS/JS admin panel.

## Deploy
Nginx orqali `/var/www/innovateit-frontend` papkasiga joylashtiriladi.
Backend API: `/api` (relative URL — nginx proxy orqali)
